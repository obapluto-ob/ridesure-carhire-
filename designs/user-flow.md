# 🗺️ User Flow – RideSure Car Hire App

## 👤 Customer Flow
1. Home Page
2. Login / Register
3. View All Cars
4. Click on a Car to View Details
5. Click "Book Now"
6. Fill Booking Form
7. See Booking Confirmation

---

## 👨‍💼 Admin Flow
1. Admin Login Page
2. Admin Dashboard
3. Add a New Car (form)
4. View List of All Cars
5. View All Bookings
